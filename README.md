此程序按照我个人喜好进行编写。我是手机重度用户，目前OLED屏幕越来越普遍，各大厂商的深色模式做的也越来越完善，
尤其是我非常追求统一程度，所以在正式版中写了强制跟随系统变换深色模式 来达到桌面的统一，不至于晚上回到桌面上亮瞎眼。

资产小组件也是我使用以来觉得用到最频繁的一些信息，相比于滚动红包和京豆收益柱状图来说使用效率更高，更适合自己查看。在此基础上增加了一些自定义选项供更多的人来自定义显示内容。
